# Screenshots

Add your screenshots here after running the app locally.

Suggested filenames (referenced in the main README):
- `dashboard.png` — the main dashboard with stat cards and alerts
- `inventory.png` — the inventory table with filters
- `optimize.png` — the optimization page with reorder suggestions

## How to take good screenshots

1. Run `npm run seed` to populate the DB with realistic data
2. Open `http://localhost:3000` in Chrome
3. Use the browser's built-in screenshot tool or a tool like [CleanShot X](https://cleanshot.com) (Mac) / [Greenshot](https://getgreenshot.org) (Windows)
4. Aim for 1400×900px or similar widescreen resolution
5. Save as PNG and drop them in this folder
